# CORD-19 Frameworks Assignment

## 📖 Overview
This project analyzes the **CORD-19 metadata dataset** and builds a **Streamlit app** for interactive exploration of COVID-19 research papers.

## ⚙️ Tools
- Python 3.7+
- pandas
- matplotlib / seaborn
- Streamlit
- wordcloud

## 🚀 Running the Project
1. Clone this repo
2. Install dependencies:
   ```bash
   pip install -r requirements.txt

Place metadata.csv inside data/

Run Jupyter Notebook (analysis.ipynb) or Python script:

python analysis.py


Launch the Streamlit app:

streamlit run app.py

📊 Features

Publications by year

Top publishing journals

Word cloud of paper titles

Interactive year filter

Data preview
